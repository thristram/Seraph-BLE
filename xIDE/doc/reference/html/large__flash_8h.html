Detailed Description

This module provides access to large (> 512-kbit) SPI Flash devices. Currently the following devices are supported:

Adesto AT25DF011 (1-Mbit) Macronix MX25L4006E (4-Mbit)

Two PIOs must be allocated to permit access to the device. See CS-324434: Interfacing Large Serial Flash and EEPROM Application Note for more details.
