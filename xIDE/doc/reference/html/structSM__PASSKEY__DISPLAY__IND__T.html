Detailed Description

This event is raised during Passkey Pairing and requires handling by a call to SMPasskeyDisplayed() or SMPasskeyInputNeg(). If the Application displays an alternative passkey, SMPasskeyInput() is used instead of SMPasskeyDisplayed() to inform the SM stack.
