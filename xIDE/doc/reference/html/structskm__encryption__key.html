Detailed Description

Security Key Manager Encryption Key information.

This structure defines all the information to be stored for an encryption key for a single device. This information is provided to the application by the Security Manager when a key is generated via the SM_KEYS_IND message, and may be requested via the SM_KEY_REQUEST_IND message.
