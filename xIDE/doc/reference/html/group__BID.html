Detailed Description

Information about application build, firmware build, and Bluetooth version supported.

Define Documentation

#define BUILD_IDENTIFIER_STRING   "bdkSDK2_6_1_275_16..."

Firmware abbreviated ID string.

Not currently used by the FW but retained to keep build system happy.

#define BUILD_IDENTIFIER_STRING_FULL   "bdk_SDK_2_6_1_275_________1606221138"

Firmware full ID string.

This is stored in ROM constant data and can retrieved by various means (e.g. direct access via SLT, UCI command, etc.) Non-release builds will use a generic "Unknown_TIMESTAMP" string rather than a formal label like below.
