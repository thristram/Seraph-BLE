Enumeration Type Documentation

enum sys_event_id

System event codes.

Enumerator:
sys_event_wakeup 

The system was woken by an edge on the WAKE pin. See wakeup_data for associated data.

sys_event_battery_low 

The system battery voltage has moved above or below the monitoring threshold. See battery_low_data for associated data.

sys_event_pio_changed 

One or more PIOs specified by PioSetEventMask() have changed input level. See pio_changed_data for associated data.

sys_event_pio_ctrlr 

An event was received from the 8051 PIO Controller. See pio_ctrlr_data for associated data.
