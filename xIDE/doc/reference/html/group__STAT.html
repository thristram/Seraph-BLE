Status Groups

Within the complete set of status codes defined across the system, there are groups of codes corresponding to specific functional areas. For example, all I2C status codes are grouped together and given a common prefix. These constants define the "base" code for each group of codes. These base codes are then used in the definition of the status enumerated type.

enum sys_status

HCI and extended system-wide status codes.

Please refer to the BlueTooth specifications V4.0, volume 2, part D for details of the HCI error codes in the range 0x00 - 0x3F.

The extended error codes (those above 0x0100) are documented here.
