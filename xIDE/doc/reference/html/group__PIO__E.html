Edge Capture Mode
[Programmable Input/Output]

Edge Capture Mode PIO functions

Functions for controlling the behaviour of PIOs which have a mode of pio_mode_edge_capture.

These functions only allow for overall control of all PIOs in pio_mode_edge_capture mode. There is no way to enable or disable edge capture on an individual PIO except by changing the mode of the PIO.
