void PioCtrlrClock ( bool  fastest_available )

Select the clock source for the PIO Controller.

The PIO Controller normally runs off the 32kHz clock. This function can be used to instead request the use of the fastest available clock, in which case the PIO Controller will run off the 16MHz clock when that clock is running (which is any time the chip is not in Deep Sleep).

NOTE: if you need a guaranteed 16MHz clock source, then as well as calling this function you must also call SleepModeChange() and set the sleep mode to either sleep_mode_never or sleep_mode_shallow to ensure that the 16MHz clock is not turned off when the radio is idle.

Parameters:
fastest_availableIf TRUE then the firmware will use the fastest available clock source for the PIO Controller. If FALSE, then the PIO Controller will always use the 32kHz clock.
Returns:
Nothing.
