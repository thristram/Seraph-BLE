User Guides

Helpful guides on frequently used SDK tools, features and the CSR µEnergy Bluetooth USB dongle driver:

xIDE User Guide

Describes xIDE for CSR µEnergy, the Integrated Development Environment for developing application software to run on CSR µEnergy chips.

GATT Database Generator User Guide

Describes how to use the GATT Database Generator utility.

Bluetooth USB Dongle Driver Installation User Guide

Describes how to install the USB driver required by CSR µEnergy applications.

Library Project Demonstration User Guide

Demonstrates how to create and link libraries using xIDE for CSR µEnergy.

Firmware Library Documentation

Long Term Key (LTK) Application Note

Demonstrates the Non Volatile Memory (NVM) storage and retrieval functionality using the Long Term Key as an example.

Keyless Entry System (KES) User Guide

Describes how to use the custom CSR automotive KES profiles and demonstrator applications.

Interfacing Large Serial Flash and EEPROM Application Note

Describes how to access large serial Flash and EEPROM memory parts.

Over-The-Air Update Documentation

The CSR Over-The-Air Update system: documents for application developers.

Large NVM Example Library

Describes an implementation example of how to extend the Firmware Library API to access the NVM Store at addresses above 64 KB.

Describes the Bluetooth Smart profile-based example applications included in the SDK. Each application note details the design, operation, and configuration of the on-chip application, and information on how to measure current consumption.

Describes the example applications included in the SDK that provide a simple demonstration of how to use the CSR µEnergy device's on-board functionality and external interfaces.
