Peripheral and Interface Example Applications

On-chip Example Applications

Analogue Input/Output Example

Demonstrates using Analogue Input/Output (AIO) ports.

Digital Programmable Input/Output Example

Demonstrates using Digital Programmable Input/Output (PIO) ports to change LED states and to read the button status.

Pulse Width Modulation (PWM) Example

Demonstrates using the Pulse Width Modulation (PWM) module to output variable duty cycle and flash an LED.

Quadrature Decoder Example

Demonstrates using the Quadrature Decoder.

Timers Example

Demonstrates timers and timer routines.

UART Input/Output Example

Demonstrates Input/Output over the Universal Asynchronous Receiver/Transmitter (UART) by implementing a simple UART loopback.

Interface Applications

SPI Master Example

Demonstrates using the PIO Controller as a SPI master

SPI Slave Example

Demonstrates using the PIO Controller as a SPI slave

SPI Test Setup

Demonstrates using the SPI master and SPI slave examples to communicate between two devices over a SPI bus
